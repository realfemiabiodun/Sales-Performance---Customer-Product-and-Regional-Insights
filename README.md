# Sales-Performance Customer-Product-and-Regional-Insights
##Excel-based Sales Performance Dashboard analyzing customer, product and regional insights, developed during the Easy Technologies Academy Data Analytics Internship.
##Project Overviews
The project is an interactives Sales Performance Dashboard built on Microsoft Excel to uncover meaningful insights from the sales data and communicate them through an interactives Excel dashboard. It address business performance, customer behaviour, and regional trend using visual storytelling to support data-driven decisions.
